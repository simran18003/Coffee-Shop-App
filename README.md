# Coffee Shop App

Coffee Shop App is an Android application developed using **Kotlin** that represents a digital coffee shop experience.  
The app focuses on clean UI design, smooth navigation, and proper Android architecture practices.

This project is created to strengthen Android fundamentals such as activities, adapters, RecyclerView, and MVVM architecture.

---

##  App Features

- Splash screen with branding  
- Home screen displaying coffee categories and popular items  
- Detailed product screen  
- Add to cart functionality  
- Cart management screen  
- Smooth navigation between screens  
- Responsive UI for different screen sizes  

---

##  Architecture Used

The project follows **MVVM (Model–View–ViewModel)** architecture to maintain separation of concerns and scalability.
